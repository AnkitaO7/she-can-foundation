# She Can Foundation Website

A simple responsive NGO webpage created as part of the frontend development internship selection task for She Can Foundation using HTML, CSS, and JavaScript.

## Features

* Responsive design
* Dark mode toggle
* Hero section
* About Us section
* Footer with social links
* Popup message on button click

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Purpose

This website was created for the frontend development internship selection process and is intended for learning/demo purposes only.
